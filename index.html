JSONファイル名の命名規則の違い

v1.0 のファイル名

calendar_events_[YYYY][MM][DD]_[HH][mm][ss].json

例: calendar_events_20250710_183015.json

v2.0 のファイル名

Calendar-[CALENDAR_VERSION]_[YYYY]_[MM]_[DD]_[HH]_[mm]_[ss].json

例: Calendar-2.0_2025_07_10_18_30_15.json

v3.0 のファイル名

Calendar-3.0_[YYYY]_[MM]_[DD]_[HH]_[mm]_[ss].json

例: Calendar-3.0_2025_07_10_18_30_15.json

v1.0 と v2.0（v3.0）はファイル名やJSON内容が異なり互換性はありません。各バージョンに合わせて操作してください。

⚠️ 注意: v1.0 と v2.0（v3.0）を同時に使用すると、片方のカレンダー表示が崩れることがあります。
片方のみの利用を推奨します。

Differences in JSON Filename Format

v1.0 format

calendar_events_[YYYY][MM][DD]_[HH][mm][ss].json

Example: calendar_events_20250710_183015.json

v2.0 format

Calendar-[CALENDAR_VERSION]_[YYYY]_[MM]_[DD]_[HH]_[mm]_[ss].json

Example: Calendar-2.0_2025_07_10_18_30_15.json

v3.0 format

Calendar-3.0_[YYYY]_[MM]_[DD]_[HH]_[mm]_[ss].json

Example: Calendar-3.0_2025_07_10_18_30_15.json

The filenames and JSON content differ between v1.0 and v2.0 (v3.0), so they are not compatible.
Please use each version separately.

⚠️ Note: Using v1.0 and v2.0 (v3.0) simultaneously may cause display issues.
We recommend using only one version at a time.

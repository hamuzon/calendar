JSONファイル名の命名規則の違い

v1.0 のファイル名

calendar_events_[YYYY][MM][DD]_[HH][mm][ss].json

例: calendar_events_20250709_150230.json

v2.0 のファイル名

Calendar-[CALENDAR_VERSION]_[YYYY]_[MM]_[DD]_[HH]_[mm]_[ss].json

例: Calendar-2.0_2025_07_09_15_02_30.json

v1.0 と v2.0 はファイル名とJSON内容が異なり互換性はありません。
各バージョンに合わせて操作してください。

⚠️ 注意: v1.0とv2.0を同時に使用すると、片方のカレンダー表示が崩れることがあります。
片方のみの利用を推奨します。

Differences in JSON Filename Format

v1.0 format

calendar_events_[YYYY][MM][DD]_[HH][mm][ss].json

Example: calendar_events_20250709_150230.json

v2.0 format

Calendar-[CALENDAR_VERSION]_[YYYY]_[MM]_[DD]_[HH]_[mm]_[ss].json

Example: Calendar-2.0_2025_07_09_15_02_30.json

The filenames and JSON content differ between v1.0 and v2.0, and they are not compatible.
Please operate each version separately.

⚠️ Note: Using v1.0 and v2.0 simultaneously may cause display issues in one of the calendars.
We recommend using only one version at a time.

📅 カレンダー選択 📅

以下のバージョンから選んでください。

🗓️ v1.0 🧾 📆 v2.0 🖋️

JSONファイル名の命名規則の違い

v1.0 のファイル名

calendar_events_[YYYY][MM][DD]_[HH][mm][ss].json

例: calendar_events_20250709_150230.json

v2.0 のファイル名

Calendar-[CALENDAR_VERSION]_[YYYY]_[MM]_[DD]_[HH]_[mm]_[ss].json

例: Calendar-2.0_2025_07_09_15_02_30.json

v1.0 と v2.0 はファイル名とJSON内容が異なり互換性はありません。
各バージョンに合わせて操作してください。

⚠️ 注意: v1.0とv2.0の両方の予定ファイルを同時に公開すると、表示が崩れる可能性があります。
ただし、どちらか一方のみを設置している場合は、問題が起きない場合もあります。

JSON Filename Format Differences

v1.0 Format

calendar_events_[YYYY][MM][DD]_[HH][mm][ss].json

Example: calendar_events_20250709_150230.json

v2.0 Format

Calendar-[CALENDAR_VERSION]_[YYYY]_[MM]_[DD]_[HH]_[mm]_[ss].json

Example: Calendar-2.0_2025_07_09_15_02_30.json

The JSON format and filenames are different between v1.0 and v2.0 and are not compatible.
Please operate within each version accordingly.

⚠️ Note: If both v1.0 and v2.0 event files are published simultaneously, display issues may occur.
However, using only one version at a time may avoid such issues.
